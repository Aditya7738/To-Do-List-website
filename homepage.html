~* About website *~

  • This website provide two types of account. One for Student and second for Other Peoples.

  • For student, I have provided special type of task-table which inspired by The Eisenhower Matrix. In which, system will guide to user which task should perform right now.

  • For other peoples, I have provided Task-List in which they can not only add task but also can set start date & time, due date & time, priority, status, remainder and edit, update, delete task. They will get remainder of task for the day via Gmail.
